Macros

When you select the Macros tab in the main window, DHCP Manager displays a list of macros in the server's dhcptab.

The macros are listed alphabetically by name. Macros that include other macros are identified by a folder icon. To view the included macro(s), click the open button to the left of the folder icon. Click a macro icon to view a list of options assigned to this macro, and their values.

You can create a new macro, modify, copy, or delete an existing macro by selecting it and clicking the appropriate entry in the Edit menu.

For more information about macros and how to use them, see About Macros and Options.

The column settings are described below.

Macro The name of the macro. A macro may include one or more macros.
Option The name of the option(s) included in the selected macro.
Value The value of the option.
