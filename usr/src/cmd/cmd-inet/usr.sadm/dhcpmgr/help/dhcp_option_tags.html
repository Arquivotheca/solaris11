Standard Options

The standard DHCP options are defined by a short name (or tag), which is a mnemonic. The names are listed below with their code number, name, data type, and description.

Code Name Type Description
1 Subnet IP Subnet mask, dotted Internet address
2 UTCoffst Number Coordinated universal Number offset (seconds)
3 Router IP List of routers
4 Timeserv IP List of RFC-868 servers
5 IEN116ns IP List of IEN-116 name servers
6 DNSservIP List of DNS name servers
7 Logserv IP List of MIT-LCS UDP log servers
8 Cookie IP List of RFC-865 cookie servers
9 Lprserv IP List of RFC-1179 line printer servers
10 Impress IP List of Imagen Impress servers
11 Resource IP List of RFC-887 resource location servers
12 Hostname Boolean Return client's host name, value from hosts database
13 Bootsize Number Number of 512 octet blocks in boot image
14 Dumpfile ASCII Path where core image should be dumped
15 DNSdmain ASCII DNS domain name
16 Swapserv IP Client's swap server
17 Rootpath ASCII Client's root path
18 ExtendP ASCII Extensions path
19 IpFwdF Number IP forwarding enable/disable (1/0)
20 NLrouteF Number Non-local source routing
21 PFilter IP Policy filter
22 MaxIpSiz Number Maximum datagram reassembly size
23 IpTTL Byte Default IP time to live (1=< x <=225)
24 PathTO Number RFC-1191 Path MTU aging timeout

26 MTU Number Interface MTU, (x <=68)
