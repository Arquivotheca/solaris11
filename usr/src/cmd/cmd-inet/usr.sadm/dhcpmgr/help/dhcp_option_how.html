How To: Options

View Options

Choose the Options tab in the main window.

DHCP Manager displays a list of options in the DHCP server's dhcptab.

Create a New Option

  1. Choose the Options tab in the main window.
    DHCP Manager displays a list of options in the DHCP database.

  2. Choose Create from the Edit menu.
    The Create Option dialog box opens.

  3. Enter the desired settings for the new option.
    You can enter the following settings in the Create Option dialog box:
    • Option name
    • Category
    • Client Class
    • Code
    • Data type
    • Granularity
    • Maximum

  4. Click OK to accept your settings.
    The new option definition is stored in the dhcptab, and is available to include in a macro.

Duplicate an Option Definition

  1. Choose the Options tab in the main window.
    DHCP Manager displays a list of options.

  2. Choose the option definition you want to duplicate.

  3. Choose Duplicate from the Edit menu.
    The Duplicate Option Properties dialog box is displayed. The fields display the settings of the selected option.

  4. Enter a name for the new option.

  5. Modify the option definition settings as desired, and then click OK.
    The new option definition is stored in the dhcptab.
Modify an Option Definition

  1. Choose the Options tab in the main window.
    DHCP Manager displays a list of options.

  2. Choose the option definition you want to modify.

  3. Double-click the option or choose Properties from the Edit menu.
    The Option Properties dialog box is displayed.

  4. Modify the option definition settings as desired, and then click OK.
    The new settings are updated in the dhcptab.
Delete an Option Definition

  1. Choose the Options tab in the main window.
    DHCP Manager displays the list of defined options.

  2. Choose the option definition you want to delete.

  3. Choose Delete from the Edit menu.
    The option definition is deleted from the dhcptab database.
