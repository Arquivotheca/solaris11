Configuring a BOOTP Relay Agent

If you start the DHCP Manager for a system that has not been configured as either a DHCP server or as a BOOTP relay agent, you can choose how to configure the system.

If you choose to configure the system as a BOOTP relay agent, the Configure BOOTP Relay dialog box is displayed.

DHCP Servers Enter the IP address(es) or host name(s) of the DHCP servers to which this relay agent will forward requests for configuration.
 

After initial configuration, use the Modify option in the Service menu to configure additional options, such as the number of relay hops and which network interfaces to monitor.
