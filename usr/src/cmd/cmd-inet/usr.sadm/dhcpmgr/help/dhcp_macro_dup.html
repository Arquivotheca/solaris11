Duplicate Macro

The Duplicate Macro dialog box provides a simple way for you to create a new macro by copying some or all of the settings of an existing macro.

Refer to About Macros and Options, for more information about defining DHCP macros.

The settings in the Duplicate Macro dialog box are described below.

Name Specify a name for the new macro. Macro names can be a maximum of 128 alphanumeric characters.
Option Name of the options and/or other macros included in the macro. You can add or delete options from this list. To display a list of defined options, right-click in the Options field. Options are displayed by category and type.
Move Up/Move Down Click Move Up or Move Down to change the processing order of options and nested macros. The processing order is significant, as options and macros processed last supersede conflicting values in those that are processed first.
Delete Select an option and click Delete to remove it from the macro.

Notify DHCP Server of Change: Check the box if you want the DHCP server to reload the dhcptab after the changed macro information is written. Normally, you want the server to reload the dhcptab so that clients start receiving the new configuration data immediately. However, in some cases, you may wish to delay this reload. For example, you may need to modify several macros in order for configurations to be correct.
