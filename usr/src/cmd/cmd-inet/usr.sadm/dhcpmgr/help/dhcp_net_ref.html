Networks

The DHCP server is connected to one or more networks. One network is configured when the DHCP server is first configured.

To add one or more networks to those monitored by the DHCP server, choose the Network Wizard, available from the Edit menu, when the Address view is displayed.

To remove configured networks from those monitored by the DHCP server, choose Delete Networks from the Edit menu .
