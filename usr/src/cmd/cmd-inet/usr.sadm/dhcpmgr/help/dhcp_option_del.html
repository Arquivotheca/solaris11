Delete Option

Click OK to remove the selected option from the dhcptab.

Notify DHCP Server of Change: Check the box if you want the DHCP server to reload the dhcptab immediately after the data is written. Normally, you will want this to happen. However, if you are making multiple changes, you may choose to wait so that the server reloads only after all the changes are written.
