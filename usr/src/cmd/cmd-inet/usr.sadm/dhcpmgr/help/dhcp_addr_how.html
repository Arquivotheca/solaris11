How To: Addresses

View IP Addresses

  1. Choose the Addresses tab in the main window.

  2. Choose a network from the Network list.
    DHCP Manager displays a list of client records in the current network, listed by IP address.

To change the way the addresses data is displayed, click in the column header. For example, click the Server column heading to view the records alphabetically in ascending order by server name. Hold down the shift key and click the Server column heading to view the records listed by server in descending order.

Create a New IP Address

  1. Choose the Addresses tab in the main window.

  2. Choose a network from the Network list.
    DHCP Manager displays a list of client records in the current network, listed by IP address.

  3. Click Create in the Edit menu.
    The Create Address dialog box opens.

  4. Enter the desired settings for the new address.
    You can enter the following settings in the Address screen:
    • IP address - the new address
    • Client name
    • Owning server
    • Configuration macro - scroll for available macro
    Click Lease to enter further settings:
    • Client ID
    • Lease status
    • Lease expiration

  5. Click OK to exit the dialog.

Add Multiple IP Addresses

  1. Choose the Addresses tab in the main window.

  2. Choose a network from the Network list.
    DHCP Manager displays a list of client records in the current network, listed by IP address.

  3. Click Address Wizard in the Edit menu.
    The Address Wizard starts.

  4. Follow the directions to add one of more addresses to the selected network.
    You can enter values for the following settings:
    • Owning server
    • Number of addresses to add
    • Starting IP address
    • Whether to generate host names for clients
    • Configuration macro
    • Lease status
    • Lease type

  5. Click OK to exit the dialog box.

Duplicate an IP Address

  1. Choose the Addresses tab in the main window.

  2. Choose a network from the Network list.
    DHCP Manager displays a list of client records in the current network, listed by IP address.

  3. From the list of addresses, select a client record that you want to copy.

  4. Click Duplicate from the Edit menu.
    The Duplicate Address dialog box opens. Some of the fields have settings that duplicate those of the selected client record.

  5. Enter the new IP address in the IP Address field.
    You can change any of the settings listed for this address.

  6. Click OK to exit the dialog box.

Modify an IP Address

  1. Choose the Addresses tab in the main window.

  2. Choose a network from the Network list.
    DHCP Manager displays a list of client records in the current network, listed by IP address.

  3. Select the client record you want to modify, and then choose Properties from the Edit menu (or double-click in the table).
    The Address Properties dialog box is displayed.

  4. Enter the desired settings for the addresses, and then click OK to exit the dialog.

Modify Multiple IP Addresses

  1. Choose the Addresses tab in the main window.

  2. Choose a network from the Network list.
    DHCP Manager displays a list of client records in the current network, listed by IP address.

  3. Select the client records you want to modify, and then choose Properties from the Edit menu.
    You can select a range of addresses by holding down the Shift key as you highlight your choices.

    To select individual addresses, hold down the Control key and select the addresses you want to modify.

    The Modify Multiple Addresses dialog box is displayed.

  4. Edit this screen to change one or more settings.

  5. Click OK to exit the dialog.

Release IP Addresses

  1. Choose the Addresses tab in the main window.

  2. Choose a network from the Network list.
    DHCP Manager displays a list of client records in the current network, listed by IP address.

  3. Select one or more addresses to release, and then choose Release from the Edit menu.
    The Release Address dialog box opens.

  4. Click OK to release the selected addresses.

Delete IP Addresses

  1. Choose the Addresses tab in the main window.

  2. Choose a network from the Network list.
    DHCP Manager displays a list of client records in the current network, listed by IP address.

  3. Select one or more addresses to delete, and then choose Delete from the Edit menu.

  4. Click OK to delete the selected addresses.
