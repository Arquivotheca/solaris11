Release Addresses

The Release Addresses dialog box lets you reclaim one or more addresses in the selected DHCP network. Reclaiming the address clears the Client ID and Lease fields.

Click OK to release the selected addresses.

Note: Reclaiming the address will not clear the Reserved field. To make a reserved address available to another client, modify the address properties using the Properties dialog box from the Edit menu. Uncheck the box marked Reserved, then select Dynamic and enter a lease expiration date.
